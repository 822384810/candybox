# candybox
项目名称：糖果盒

# 组件列表
1.注册中心：Spring Cloud Eureka

2.网关：Spring Cloud Gateway

3.配置中心：Spring Cloud Config

4.流量控制：Alibaba Sentinel

5.Api文档：Swagger3

6.缓存：Redis（使用时注意修改配置文件中的主机IP，端口、密码）
